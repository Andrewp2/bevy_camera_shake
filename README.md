Third party camera shake plugin for Bevy

Based off of https://www.youtube.com/watch?v=tu-Qe66AvtY

Uses a "trauma" based system to vary the amount of camera shake.

Compatibility:

| bevy_camera_shake version | Bevy version |
| ------------------------- | ------------ |
| 0.1.0                     | 0.8.1        |
| 1.0.0                     | 0.9.0        |
| 1.1.0                     | 0.10.0       |
| 2.0.0                     | 0.11.x       |
